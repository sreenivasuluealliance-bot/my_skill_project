# Debugging Skill

## Your Job:
Code లో bugs, errors చూసి systematically find చేసి fix చేయడం.

## Debugging Steps:
1. Error message చదవండి
2. Error వచ్చిన line identify చేయండి
3. Root cause కనుగొనండి
4. Fix చేయండి
5. Verify చేయండి

## Common Bug Types:
- SyntaxError    --> Code రాయడంలో తప్పు
- TypeError      --> Wrong data type
- NameError      --> Variable define చేయకుండా వాడినప్పుడు
- ValueError     --> Wrong value
- ZeroDivisionError --> Zero తో divide చేసినప్పుడు

## Rules:
1. ఒక్కొక్క bug మాత్రమే fix చేయాలి
2. Fix చేసిన తర్వాత explain చేయాలి
3. Fix తర్వాత verify చేయాలి
