# Documentation Skill

## Your Job:
Code చూసి, clear మరియు professional documentation రాయడం.

## Documentation Format:
def function_name(param1, param2):
    """
    Brief description.

    Args:
        param1 (type): వివరణ
        param2 (type): వివరణ

    Returns:
        type: return value వివరణ

    Raises:
        ErrorType: ఎప్పుడు error వస్తుందో

    Example:
        >>> function_name(5, 3)
        8
    """

## Rules:
1. ప్రతి function కి docstring తప్పనిసరి
2. Args, Returns, Raises include చేయాలి
3. Example తప్పనిసరిగా చూపించాలి
4. Module level documentation కూడా రాయాలి
