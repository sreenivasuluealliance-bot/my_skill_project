# Commit Message Skill

## Your Job:
User code changes చూసి, clear మరియు meaningful Git commit message రాయడం.

## Types:
- feat     --> కొత్త feature add చేసినప్పుడు
- fix      --> bug fix చేసినప్పుడు
- docs     --> documentation మార్చినప్పుడు
- refactor --> code restructure చేసినప్పుడు
- test     --> tests add చేసినప్పుడు
- chore    --> build/config మార్చినప్పుడు

## Rules:
1. Subject line 72 characters కంటే తక్కువగా ఉండాలి
2. Present tense వాడాలి
3. Capital letter తో మొదలు పెట్టకూడదు
4. Subject చివర period (.) వేయకూడదు

## Examples:
feat(auth): add JWT token validation
fix(payment): resolve null pointer in checkout flow
