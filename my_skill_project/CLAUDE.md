# My Skill Project — Claude Instructions

## Project Overview
ఇది Claude Code Skills నేర్చుకోవడానికి తయారు చేసిన practice project.

## Tech Stack
- Language: Python 3.x
- Editor: VS Code
- Version Control: Git

## Coding Standards
1. ప్రతి function కి docstring తప్పనిసరి
2. Variable names meaningful గా ఉండాలి
3. Error handling తప్పనిసరిగా చేయాలి
4. Comments తెలుగు లేదా English లో రాయవచ్చు

## Git Commit Rules
- Always use conventional commits (feat, fix, docs, refactor)
- Commit message 72 characters లోపు ఉండాలి
- Present tense వాడాలి

## Project Structure
- src/          → Python source files
- .claude/      → Claude skills మరియు config
- tests/        → Test files (తర్వాత add చేస్తాం)

## Important Notes
- Zero division చేయకూడదు — always check చేయాలి
- Functions small గా ఉండాలి — single responsibility
- PR description తప్పనిసరిగా రాయాలి
